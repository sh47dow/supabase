### `match()`

<div style={{display: props.verb == 'get' ? 'block' : 'none'}}>

<!-- prettier-ignore -->
```js {3}

  .get(tableName)
  .match(filterObject)
```

</div>
<div style={{display: props.verb == 'patch' ? 'block' : 'none'}}>

<!-- prettier-ignore -->
```js {3}

  .patch(tableName, data)
  .match(filterObject)
```

</div>
<div style={{display: props.verb == 'delete' ? 'block' : 'none'}}>

<!-- prettier-ignore -->
```js {3}
supabase
  .delete(tableName)
  .match(filterObject)
```

</div>

Finds rows that exactly match the specified filterObject.

<Collapsable title="Toggle Example">

<div style={{display: props.verb == 'get' ? 'block' : 'none'}}>

<!-- prettier-ignore -->
```js

  .get('countries')

```

</div>
<div style={{display: props.verb == 'patch' ? 'block' : 'none'}}>

<!-- prettier-ignore -->
```js

  .patch('countries', { name: 'Middle Earth' })
  .match({ 'continent': 'Asia' })
```

</div>
<div style={{display: props.verb == 'delete' ? 'block' : 'none'}}>

<!-- prettier-ignore -->
```js
supabase
  .delete('countries')
  .match({ 'name': 'Mordor' })
```

</div>

</Collapsable>


##### filterObject `:object`
An object of `{ 'columnName': 'criteria' }`
