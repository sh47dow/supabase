#### General form
<!-- prettier-ignore -->
```js {3} 
supabase
  .from(tableName)
  .ova(columnName, filterArray)
```

Finds all rows whose array value on the stated columnName overlaps with the specified filterArray. Equivalent of `filter(columnName, 'ova', criteria)`.

#### `columnName: string`
Name of the database column.

#### `filterArray: array`
Value to compare to.
