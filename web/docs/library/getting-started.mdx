## Installing

```bash
npm install --save @supabase/supabase-js
# OR
yarn install @supabase/supabase-js
```


## Initializing

```js
import { createClient } from '@supabase/supabase-js'

// Create a single supabase client for interacting with your database 
const supabase = createClient("https://xyzcompany.supabase.io", "1a2b-3c4d-5e6f-7g8h");
```



## Reference

### `createClient()`

```js 
createClient('supabaseURL', 'supabaseKey', OPTIONS)
```

###### supabaseURL `:string` 
The unique Supabase URL which is supplied when you create a new project in [your project dashboard](https://app.supabase.io).

###### supabaseKey `:string`
The unique Supabase Key which is supplied when you create a new project in [your project dashboard](https://app.supabase.io).

###### OPTIONS `:object?`

```js
const options = {
  // allowBulkUpdates :boolean
  // If true, Supabase will not block bulk updates and deletes. 
  // This can be overriden on individual calls to update(), 
  // so it is recommended to keep this as false.
  allowBulkUpdates: false, 

  // allowBulkDeletes :boolean
  // If true, Supabase will not block bulk updates and deletes. 
  // This can be overriden on individual calls to delete(), 
  // so it is recommended to keep this as false.
  allowBulkDeletes: false, 
}
```
