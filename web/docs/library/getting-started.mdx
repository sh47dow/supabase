### createClient(supabaseURL, supabaseKey, options?)
This is used to initialize an instance of a `createClient` object. From here, you would be able
invoke all the other features and functions provided by Supabase.

### Method arguments
#### supabaseURL
`required` string  
Your unique Supabase URL.

#### supabaseKey
`required` string  
Your unique Supabase Key.

#### options
`optional` object
- @tbc
