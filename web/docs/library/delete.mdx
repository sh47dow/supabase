## Reference

### `from()`
<FromFunction verb='delete' />

### `delete()`
```js {3}
supabase
  .from(tableName)
  .delete(options?)
```
  
#### `options: object?` 

```js
/**
 * @param {boolean} [bulk=false] 
 * Protects against bulk updates. Set to true if you intend to perform a bulk change.
 */
const options = {
  bulk: false 
}
```
