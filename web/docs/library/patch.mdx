## Reference

### `from()`
<FromFunction verb='patch' />

### `update()`
```js {3}
supabase
  .from(tableName)
  .update(data, options?)
```
   
#### `data: object`
New record values.

#### `options: object?`   

```js
const options = {
  bulk: false // Protects against bulk updates. Set to true if you intend to perform a bulk change.
}
```
