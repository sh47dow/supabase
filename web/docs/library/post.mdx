supabase.post(tableName, data, options?)

## Method arguments
### tableName
`required` string  
Name of table in the database where data will be saved.

### data
`required` array  
Array of rows of type `object` which contain information to be saved into the selected table.


### options
`optional` object  
All available options and examples it their usage can be found [here](../restful/options).

const postUsers = async () => {

## Responses
### 201 Created
The request has succeeded. New resource created will be returned:

```json
{
  "rows": [
    { "fullName": "Richard Hendrix" },
    { "fullName": "Elrich Bachman"}
  ]
}

```
