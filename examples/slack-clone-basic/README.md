## How to use


**Install dependencies**

```sh
npm install 
```

**Start Supbase Dev**

```sh
docker-compose up -d
```

**Start the chat room**

```sh
npm run dev
```

Visit http://localhost:3000 and start slacking! Open in two tabs to see everything getting updated in realtime
